# Project: ASH VECTOR — UI Kit v0.0.2

This build introduces the first reusable ASH VECTOR interface language.

## UI Direction
- Classified military terminal
- CRT scanlines and glitch effects
- Ash gray, warning red, neon cyan, toxic green
- Database-style panels instead of generic fantasy menus

## In-game naming rules
- Characters = Operators
- Enemies = Anomalies
- Skills = Combat Protocols
- Inventory = Equipment Database
- Saves = Vector Archives
- XP/Leveling = Synchronization
- Dungeons = Sectors / Fractures
