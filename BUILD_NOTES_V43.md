# Build Notes v43 — Main Menu Recovery Fix

Fixes the launch/menu screen getting stuck because the menu buttons were pushed out of the fixed-height fullscreen panel.

## Changes
- Main menu card can scroll internally instead of clipping buttons.
- Reduced title/logo scale on shorter screens.
- Added a visible quick-start hint.
- Pressing Enter from the main menu now starts the game.
- Keeps full-window layout and no body scrolling.
