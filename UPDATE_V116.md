# ASH VECTOR Update V116

## Twelve Stage Expansion Pass

This is a bigger content patch adding six more playable stages and scaling the route toward the planned 20-stage / level-99 endgame.

### New playable stages
- F-007 // Cinder Express Yard — Req Lv 30
- F-008 // Flooded Data Vault — Req Lv 36
- F-009 // Rust Orchard — Req Lv 42
- F-010 // Blacksite Observatory — Req Lv 49
- F-011 // Cryo Basilica — Req Lv 56
- F-012 // Ash Crown Citadel — Req Lv 63

### Major changes
- Added 6 larger 60x36 maps with more rooms, longer routes, and more locked boss/security areas.
- Added each new stage to normal progression and Playtest Console automatically through STAGE_DEFS.
- Added different anomaly names and a different boss for every new stage.
- Added new boss cores and rewards for F-007 through F-012.
- Added intro, terminal, lore, boss intro, boss defeated, chapter clear, and Fermilat story scenes for all six new stages.
- Added Fermilat side favors for all six new stages.
- Added Story Archive entries for Chapters 7-12.
- Expanded Protocol Challenges with Twelve-Stage Route and Road to 20 Stages goals.
- Main player level is now capped at 99.
- Level requirements now scale toward the planned 20-stage endgame route.
- Route beacon now understands opened boss/security doors better.
- Bumped visible build to v0.9.26 and cache links to ?v=116.
