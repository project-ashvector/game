## Added
- Added `Vector Cell`, a new consumable that restores EP.
- New games start with 2 Vector Cells.
- Existing saves receive 2 Vector Cells once after loading.
- Vector Cells can be bought from the Gear Terminal / vendor.
- Chests can drop Vector Cells.
- Normal anomaly fights can drop Vector Cells.
- Boss fights reward 2 Vector Cells.
- Added a battle button: **Use Vector Cell**.
- Added field hotkey: **R** uses a Vector Cell outside battle.
- Added **WASD movement** alongside arrow-key movement.

## Hotkey note
- `D` is now right movement as part of WASD.
- Anomaly Database hotkey moved to `V`.
