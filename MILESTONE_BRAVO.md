# Milestone Bravo — Chapter 1 Playtest Sprint

This build focuses on making the first chapter playable before GitHub deployment.

## Included in v15
- Objective tracker visible during gameplay
- Mission progress panel
- Mutable dungeon state: opened caches and defeated anomalies disappear
- First combat tutorial transmission
- Recovery terminal save event
- Boss intro dialogue
- Boss defeat transmission
- End-of-chapter victory screen
- Save file now preserves dungeon state

## Chapter 1 Flow
1. Start new operation.
2. Read opening AVOS transmission.
3. Reach Recovery Terminal.
4. Clear three Ashborn Revenant anomalies.
5. Defeat Ashborn Revenant Lord.
6. Reach exit tile.
7. Trigger Chapter Complete screen.
