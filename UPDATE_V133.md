# ASH VECTOR Update V133

## Skill Level-Up Requirements Pass

This patch improves field skilling feedback and makes resource training scale by zone.

### Changes
- Added medium on-screen skill level-up notification.
- Level-up notification shows for 3 seconds.
- Notification includes the skill icon, skill name, and new level reached.
- Different collected items now give different XP amounts.
- Skilling XP scales by item tier and zone/stage.
- Skilling objects now have level requirements that scale upward by zone.
- Locked skilling nodes show the required level when nearby.
- If a rare bonus item drops from a node, it can also grant XP if you meet its requirement.
- Bank item details now show skilling XP and level requirement for resource items.
- Bank slots show quick skilling requirement/XP text.
- Bumped visible build to v0.9.43 and cache links to ?v=133.
