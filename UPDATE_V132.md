# ASH VECTOR Update V132

## Mission Loot Skilling Pass

This patch adds real RPG-style progression loops to the live game.

### Changes
- Later stages now require more anomaly clears before the boss route opens.
- Objectives, route guidance, mission status, doors, exits, and victory reports now use the scaled anomaly requirement.
- Caches now scale by stage level and zone.
- Caches can drop zone-relevant materials, consumables, catalysts, and level-appropriate gear.
- Added field skilling objects that can be collected with E.
- Skilling objects train non-combat skills such as Cryptomining, Datafishing, Codecraft, Forgenetics, and System Hacking.
- Skilling objects respawn and stack items in your bag.
- XP popups now show when Sync XP or skill XP is gained.
- Inventory was upgraded into a cleaner RuneScape-style bag/bank view.
- Added a top-bar Bag button.
- Sidebar inventory now shows a compact quick bag and opens the full bank.
- Bumped visible build to v0.9.42 and cache links to ?v=132.
