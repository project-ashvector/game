# ASH VECTOR Update V118

## Expanded Layout Overhaul Pass

This patch rebuilds every currently playable stage so the game feels larger, fuller, and less repetitive without changing the actual tile PNG files.

### Changes
- Rebuilt all 12 playable stages with bigger 72x42 layouts.
- Added more open rooms, wide corridors, loops, and side branches.
- Added more locked boss/security areas using the existing door system.
- Added 8 anomaly monster tiles per stage instead of only 3 main fights.
- Kept the main objective readable: sync terminal, clear 3 anomalies, open boss wing, defeat boss, extract.
- Added different custom monster names/stats for every stage.
- Added different custom boss names/stats/rewards for every stage.
- Added unique encounter slot tables so stages use different imported monster/boss art.
- Repositioned Fermilat on all 12 expanded maps.
- Added much more decorative prop placement using existing assets only.
- Preserved clear floor/wall readability: floors remain open, walls remain solid, props only draw on open floor.
- Rebalanced stage level requirements for the planned 20-stage / Lv 99 route.
- Bumped visible build to v0.9.28 and cache links to ?v=118.
