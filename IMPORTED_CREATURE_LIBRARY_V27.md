# Imported Creature Library — v27

This build imports the provided `monsters.zip` and `bosses.zip` into the official Project: ASH VECTOR asset ID system.

## Counts
- Imported anomalies: **100**
- Imported bosses: **53**

## Folder Rules
- Normal enemies live in `assets/anomalies/an###_name/`.
- Bosses live in `assets/bosses/boss###_name/`.
- Each creature currently includes `battle.png`, `portrait.png`, and `icon.png`.
- `portrait.png` and `icon.png` are duplicated from the original battle asset for now and can be replaced later without code changes.

## Data Files
- `data/anomalies/imported_anomaly_roster.json`
- `data/bosses/imported_boss_roster.json`
- `data/registry/imported_creature_registry.json`
