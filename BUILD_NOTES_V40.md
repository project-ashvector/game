# Project: ASH VECTOR v40 — Combat UI Polish

## Summary
v40 upgrades the battle presentation so combat feels less like a developer overlay and more like an RPG battle scene.

## Changes
- Rebuilt the battle overlay layout with a fixed JRPG-style combat panel.
- Cleaned up oversized/overlapping battle title text.
- Added compact combat header with anomaly ID/name and Fracture location.
- Improved Vyra and anomaly placement in battle.
- Added polished HP/EP/focus meters.
- Added damage number popups for hits, critical hits, healing, and dodges.
- Added hit flash and screen shake effects.
- Added a victory reward panel with item icons.
- Kept the Toxic Sewers battle background from v36.
