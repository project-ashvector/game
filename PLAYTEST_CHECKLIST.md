# Project: ASH VECTOR — Playtest Checklist

Use this before uploading to GitHub.

## Boot / Menu
- [ ] AVOS boot text completes.
- [ ] Press Enter/Tap reaches main menu.
- [ ] New Game starts the opening transmission.
- [ ] Database buttons open/close correctly.

## Exploration
- [ ] Arrow keys move Vyra.
- [ ] Mobile buttons move Vyra.
- [ ] Walls block movement.
- [ ] Caches add loot and disappear.
- [ ] Recovery Terminal saves.
- [ ] Heal station restores HP.

## Combat
- [ ] Touching AN-001 starts combat.
- [ ] Four Protocol buttons work.
- [ ] Enemy HP decreases.
- [ ] Vyra HP decreases when hit.
- [ ] Defeated anomalies disappear from map.
- [ ] Loot and XP are awarded.
- [ ] Boss battle starts at B tile.
- [ ] Boss defeated unlocks the exit.

## Completion
- [ ] Exit is locked before boss defeat.
- [ ] Exit triggers Chapter Complete after boss defeat.
- [ ] Victory screen appears.
- [ ] Save/load preserves map state.
