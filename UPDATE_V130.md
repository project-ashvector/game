# ASH VECTOR Update V130

## Mobile Playability Pass

This patch cleans up phone layout, battle scaling, mobile controls, and map entity sizing.

### Changes
- Phone game view now uses the full screen without the side panels crowding the map.
- D-pad rebuilt into a clean diamond layout with the up arrow centered above left/down/right.
- Mobile buttons are smaller and less intrusive.
- Battle screen now scales for phone portrait and landscape.
- Battle sprites, attack buttons, HP bars, and text fit on small screens.
- Mobile battle quick pad stays at the bottom and no longer covers the main battle UI as much.
- Pre-battle scan modal scales better on phones.
- Player blue glow removed.
- Player, monsters, and NPCs now use the same in-map footprint/size.
- Map atmosphere keeps general highlights but removes the player-centered colored glow.
- Bumped visible build to v0.9.40 and cache links to ?v=130.
