# Project: ASH VECTOR v0.6.3 / v63

Added a real trainable skill pass and next-stage shell.

- Added RuneScape-style max-99 skill training for Attack, Strength, Defense, Health, Neurohex, Precision, and future non-combat skills.
- Skills now visibly scale stats: Attack/Strength improve damage, Defense improves mitigation, Health improves max HP.
- Added proper emblem-style skill badges in the Progression Matrix.
- Rebuilt Inventory into an operator status + loot menu showing player level, HP, EP, credits, ATK/DEF, stage, save time, and skill levels.
- Added persistent fullscreen player HUD showing Player Lv, HP, EP, and current fracture.
- Added pre-battle scan/dialog before anomaly and boss fights.
- Added F-002 stage/map shell with level requirement and unlock path after F-001 completion.
- Strengthened localStorage autosave after XP, loot, movement, upgrades, stage changes, and battle wins.
