# Project: ASH VECTOR — Game Design Document

## High Concept
A cyberpunk dark-fantasy dungeon RPG where Operators enter unstable Fractures, battle Anomalies, recover classified archives, and uncover the failed experiment known as Project: ASH VECTOR.

## Core Loop
Boot AVOS → select operation → explore Fracture → trigger events → battle Anomalies → collect loot → synchronize Operator → unlock new files and sectors.

## Tone
Dark, classified, cyberpunk, corrupted, funny, and sarcastic. The world is dangerous, but the characters talk like they are painfully aware they are stuck inside a nightmare designed by incompetent scientists.

## Terminology
- Operators: Playable characters
- Anomalies: Enemies
- Fractures: Dungeons/zones
- Protocols: Skills/attacks
- Synchronization: XP/leveling
- Archives: Save files/lore files
- Catalysts: Upgrade materials
- Credits: Currency
- AVOS: ASH VECTOR Operating System

## Launch Vertical Slice
`FR-001 Toxic Sewers` with Operator `AV-001`, a set of Anomalies, one boss, chests, dialogue, loot, and save/load.

## Long-Term Systems
- Character collection/gacha
- Operator switching
- Multiple Fractures
- Boss database
- Archive/lore system
- Crafting/upgrades
- Shops
- Daily/weekly challenges
- Endless Fracture mode
