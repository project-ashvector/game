# ASH VECTOR Update V100

## Stage 3 Detail Pass

This patch upgrades F-003 so it feels more like a handcrafted undead graveyard while keeping the easy-to-read wall and floor path system.

### Changes
- Kept the cleaner readable route and collision layout from the previous F-003 pass.
- Added more undead scenery detail to F-003 using extra props from the undead tileset pack.
- Added larger landmark props such as a lich statue, ruins, skull pile, dead-arm growth, extra graves, and heavier bone detail.
- Expanded F-003 blocked-wall art variety so edge walls look richer without making the path confusing.
- Preserved the same simple floor readability and obvious play path.
- Bumped the build to v0.9.10 and cache links to ?v=100.
