# ASH VECTOR Update V115

## Playtest Six Map Pass

This patch fixes the Playtest Console so the new maps from V114 can be loaded directly.

### Changes
- Added F-004, F-005, and F-006 to the static Playtest Console level selector.
- Added direct Playtest buttons for Load F-004, Load F-005, and Load F-006.
- Playtest level selector now rebuilds dynamically from STAGE_DEFS so future maps appear automatically.
- Direct level buttons now rebuild dynamically from STAGE_DEFS when the Playtest Console opens.
- Added useful level presets for the new map requirements: Lv 15, Lv 20, and Lv 25.
- Unlock All Levels now reports the full number of levels unlocked.
- Show Test Route now reports the current stage ID.
- Bumped visible build to v0.9.25 and cache links to ?v=115.
