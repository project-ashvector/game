# Project: ASH VECTOR — Operator Framework v0.0.4

This build adds the first real character framework for future gacha-style playable characters.

## Current Operator

- Operator ID: `AV-001`
- Name: `Vyra`
- Callsign: `Black Neon`
- Class: `Shadow Hunter`
- Element: `Void`
- Status: `ACTIVE`

## Replaceable asset paths

When final art is generated, replace these files without changing code:

```text
assets/operators/vyra/profile.png
assets/operators/vyra/portrait.png
assets/operators/vyra/battle.png
assets/operators/vyra/sprite_sheet.png
assets/operators/vyra/icon.png
```

## Data file

```text
data/operators/vyra.json
```

Future Operators should follow the same pattern:

```text
assets/operators/operator_id/profile.png
assets/operators/operator_id/portrait.png
assets/operators/operator_id/battle.png
assets/operators/operator_id/sprite_sheet.png
assets/operators/operator_id/icon.png
data/operators/operator_id.json
```

The UI already has an Operator Database screen. Open it from the main menu or from the in-game Operator Archive card.
