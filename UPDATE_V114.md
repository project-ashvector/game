# ASH VECTOR Update V114

## Three New Maps Pass

This is a larger content update that adds three full new playable fracture routes after F-003.

### New playable maps
- **F-004 // Transit Ruins**
  - Subway / rail tunnel route.
  - New route layout, terminal, lore, 3 anomalies, boss gate, boss, extraction.
- **F-005 // Glass Storm Lab**
  - Prism lab route with split chambers and storm lanes.
  - New route layout, terminal, lore, 3 anomalies, boss gate, boss, extraction.
- **F-006 // Vector Core Spire**
  - Deep core route and current end-of-chain map.
  - New route layout, terminal, lore, 3 anomalies, boss gate, boss, extraction.

### New progression
- F-003 now unlocks F-004.
- F-004 unlocks F-005.
- F-005 unlocks F-006.
- F-006 ends the current route chain and teases the next arc.
- Existing saves migrate to the expanded six-fracture route list.

### New combat content
- Added encounter slots for all 3 new maps.
- Added custom anomaly/boss definitions for the new routes.
- Added new boss rewards:
  - Transit Nexus Core
  - Prism Wound Core
  - Vector Heart Core
- Added new stage hazard identities:
  - Rail corrosion
  - Prism burn
  - Vector overload

### New story content
- Added intro, terminal, lore, boss intro, boss defeated, and chapter clear scenes for F-004, F-005, and F-006.
- Story Archive now tracks the new chapters.
- F-003 clear story now points to the new Transit Ruins route instead of “coming soon.”
- Fermilat has new stage talks and favors for F-004, F-005, and F-006.

### New map visuals
- Added visual palette/tile-pack settings for F-004, F-005, and F-006.
- Added unique floor colors, wall colors, props, and route atmosphere using existing assets.
- New maps work with Route Beacon, Objective Compass, Minimap Route, Checkpoint HUD, Enemy Intent, Victory Report, and Death Recovery.

### Protocol Challenge expansion
- Added larger route-chain challenges:
  - Deep Route Scout
  - Six-Fracture Chain
  - Core Breaker+
