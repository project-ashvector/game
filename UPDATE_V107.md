# ASH VECTOR Update V107

## Nav Assist Settings Pass

This patch gives testers and players control over the new navigation help.

### Changes
- Added Configuration toggles for Route Beacon, Objective Compass, and Minimap Route.
- Route Beacon can now be hidden without disabling the objective system.
- Objective Compass can now be hidden from the screen.
- Minimap Route pathline can now be hidden separately.
- Mission Briefing now reports when the route beacon is hidden.
- Added a Playtest Console button to toggle all navigation assists at once.
- Bumped visible build to v0.9.17 and cache links to ?v=107.
