# Project ASH VECTOR - V190 Mobile Lockdown Scaling Pass

This update focuses on making the new Vector Lockdown content fit better on phones and small touch screens.

## Changed

- Added responsive CSS for the Vector Lockdown HUD.
- Lockdown HUD now becomes compact on phones and touch devices.
- Countdown warning text is shorter on mobile.
- Active lockdown status text is shorter on mobile.
- Buff/debuff history shows fewer chips on mobile so it does not cover the screen.
- Roll popup is resized for phone screens.
- HP bar and stat chips are tightened for small screens.
- Sealed arena labels draw fewer repeated tiles on mobile to reduce clutter.
- Mobile arena radius is slightly smaller so the lockdown box does not dominate phone view.
- Boot build header stacks on phones so the version label does not crush the intro text.
- Updated build labels to v1.0.00.
