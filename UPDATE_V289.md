# Project ASH VECTOR — Update V289
## F-001 Cinematic Overhaul

This patch is focused on making **Fracture 001 / Forbidden Graveyard** feel more atmospheric and polished without breaking the existing game flow.

### Included changes
- Added a **cinematic F-001 backdrop layer** with distant graveyard silhouettes, moon haze, fence depth, and foreground framing.
- Added **smooth render-side player movement interpolation** so stepping between tiles feels less rigid.
- Added **field footstep / ash puff effects** during movement to make traversal feel more alive.
- Added a **one-time boss encounter cinematic card** for the F-001 boss before the normal pre-battle screen.
- Added a small **boss battle presentation boost** for stronger battle atmosphere.
- Updated the in-game build label to **v289 // F-001 CINEMATIC OVERHAUL**.
