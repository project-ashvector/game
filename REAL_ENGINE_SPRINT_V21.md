# Real Engine Sprint v21

This build is focused on making the browser project actually playable from local files or GitHub Pages.

## Test path
1. Open `index.html`.
2. Wait for AVOS boot and press Enter / Access.
3. Start `Initialize Operator`.
4. Move with arrow keys.
5. Open chests, use the recovery terminal, fight three Rust Hounds, unlock the boss gate, defeat Rust Mother, and reach the exit.
6. Test `Archive Save`, refresh, then `Continue Operation`.

## Known limitations
- Art is still production placeholder art, but filenames are final and can be replaced without code changes.
- Sprite animation is represented by static canvas drawing for now.
- Audio hooks are planned but not wired to sound files yet.
