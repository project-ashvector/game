# Build Notes v45 — Fullscreen HUD + Hotkeys

Fixes based on playtest screenshots:

- Fullscreen mode no longer hides all useful sidebar functions.
- Added keyboard shortcuts:
  - M = Map / field HUD
  - I = Inventory Database
  - D = Anomaly Database
  - O = Operator File
  - P = Progression Matrix
  - B = Mission Briefing
  - H = Help overlay
  - F = Fullscreen toggle
  - Esc = close open panels first
- Added fullscreen quickbar at the bottom of the screen.
- Added a fullscreen side HUD with HP, EP, anomaly progress, boss route, and minimap.
- Rechecked menu protocol buttons so Database, Inventory, Operator, Mission, etc. can open from main menu.
