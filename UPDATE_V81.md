# Project: ASH VECTOR v0.8.1 Patch Only

## Added
- Added a lightweight Side Quest Journal system.
- Fermilat now starts an optional favor quest on each stage after you find and talk to him.
- Each Fermilat favor tracks anomaly kills separately for F-001, F-002, and F-003.
- Completed side quests can be claimed from Fermilat or the Mission Briefing panel.
- Side quest rewards give credits, Sync XP, Anomaly Hunting XP, and stage-appropriate supplies.
- Mission Briefing now shows both the repeatable contract board and Fermilat side quest status.
- Objective tracker and Fracture Status now show side quest progress.
