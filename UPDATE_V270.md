# Project ASH VECTOR V270 – Character File View Pass

- Locked characters can now be opened/viewed in the Characters menu.
- Clicking a character card opens that character file instead of trying to immediately play as them.
- Locked character pages show portraits/profile, stats, battle moves, role/passive, shard type, and unlock progress.
- Only the Play As button remains disabled until the character is unlocked.
- No combat, save, NPC, portal, reward, controller, or asset loading logic was changed.
