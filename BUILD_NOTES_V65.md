# Build Notes v65 / v0.6.5 — Gear, Stage Rewards, Crafting

## Added
- Equipment/loadout system with slots: Weapon, Helmet, Chest, Legs, Gloves, Boots, Cape, Ring, Core, Relic.
- Gear stat bonuses now affect combat stats: ATK, STR, DEF, HP, EP, Crit, XP bonus.
- Starter gear: Vector Training Blade and Sewer Guard Vest.
- Equip/Unequip support inside Inventory Database.
- Gear drops from enemies, chests, and boss wins.
- Boss reward includes Toxic Monarch Relic.
- Gear Terminal in Inventory with vendor purchases and crafting.
- Craftable gear: Rustcore Blade and Catalyst Core.
- Vyra shard sync system in Operator Files.
- Pre-battle scan now shows Gear Power.
- Save/load includes equipment, shard sync, inventory, stage progress, and crafted/dropped gear.

## Notes
No new assets are required. Gear uses existing item art and imported weapon/armor icons.
