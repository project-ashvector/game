# Project: ASH VECTOR v0.8.5 Patch Only

## Added
- Save Slot / Backup Terminal inside Configuration.
- Three local save slots that can save, load, and delete progress independently from the active autosave.
- Export Save Code button for copying progress to another browser, phone, or backup note.
- Import Save Code button for restoring a copied save code or raw JSON save.
- Backup panel shows current stage, player level, credits, and save time.

## Why
- GitHub Pages saves are local to the browser/device.
- This makes mobile testing safer and gives players a way to transfer progress.
