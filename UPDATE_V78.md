# Project: ASH VECTOR v0.7.8 Patch Only

## Added
- Added a battle status-effect system.
- Vyra attacks can now apply effects:
  - Vector Slash can apply Bleed.
  - Phantom Dash primes an evade for the next enemy hit and can apply Shock.
  - Crimson Cascade can apply Corrosion.
  - Emergency Flex now cleanses active negative status effects.
- Enemies and bosses can now apply stage-based hazards:
  - F-001 Toxic Sewers: Poison.
  - F-002 Ash Wastes Outpost: Burn.
  - F-003 Neon Graveyard: Shock.
- Pre-battle scan now shows the current stage hazard.
- Battle HUD now displays active status effects and remaining turns.
